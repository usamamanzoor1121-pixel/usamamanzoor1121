### XDR Typhoid ML Classifier
GitHub: https://github.com/usamamanzoor1121-pixel/xdr-typhoid-ml

ML classifier predicting XDR/MDR/Susceptible phenotype in Salmonella Typhi from genomic features. Random Forest F1=0.982, XGBoost F1=0.980. SHAP analysis confirms blaCTX-M-15 and qnrS1 as defining XDR markers consistent with Pakistan XDR genotype 4.3.1.1.P1.

Stack: Python, scikit-learn, XGBoost, SHAP, NCBI Pathogen Detection

### Mpox Genomic Surveillance Pipeline
GitHub: https://github.com/usamamanzoor1121-pixel/mpox-genomic-surveillance

300 MPXV genomes classified with Nextclade v3.21. APOBEC3 signature analysis, genome-wide mutation hotspot mapping, and statistical screening for inter-clade recombinants. Recombination breakpoint mapped to ITR region with gene-level resolution. Validated against WHO-confirmed Ib/IIb recombinant case (Pullan et al. 2025).

Stack: Python, Biopython, Nextclade, scipy, NCBI Entrez

**Live: http://3.81.108.239:8502**
